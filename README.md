# 🎓 V&P English Academy - Official Landing Page

Bienvenido al repositorio oficial de la página web de **V&P English Academy**, una academia líder en la enseñanza del idioma inglés ubicada en Rivas, Nicaragua.

Esta *landing page* está diseñada estratégicamente con un enfoque en **CRO (Optimización de la Tasa de Conversión)**, **SEO local** y **Generación de Leads**, conectando directamente a los prospectos con nuestro equipo de ventas a través de WhatsApp y registrando los datos en la nube.

🌍 **Sitio Web en Vivo:** [www.vpenacademy.com](https://www.vpenacademy.com)

---

## 🚀 Características Principales (Features)

- **Captura de Leads Dinámica:** Formulario de pre-matrícula integrado nativamente con **Google Sheets** (vía Google Apps Script) y redirección automatizada a **WhatsApp** con mensajes pre-llenados.
- **Mini-Test de Nivel Integrado:** Un test interactivo de 3 preguntas programado en JavaScript puro que clasifica al usuario (Básico, Intermedio, Avanzado) y envía el resultado por WhatsApp.
- **SEO Técnico y Local:** Implementación de metadatos **Open Graph** para compartir en redes sociales y **Schema Markup (JSON-LD)** de `LocalBusiness` / `EducationalOrganization` para dominar las búsquedas en Rivas.
- **Gated Content (Lead Magnet):** Sección de recursos gratuitos optimizada donde el usuario debe contactar por WhatsApp para desbloquear el material.
- **Rendimiento Optimo:** Construido en Vanilla HTML, CSS y JS en un solo archivo para máxima velocidad, utilizando *lazy loading* para imágenes y preconexión de fuentes.
- **Diseño Responsive:** Adaptable a dispositivos móviles (Mobile-First approach), tablets y desktop.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** HTML5, CSS3 (Variables, Flexbox, CSS Grid), Vanilla JavaScript.
- **Iconografía:** FontAwesome 6.4.0 (CDN).
- **Tipografía:** Google Fonts (Montserrat).
- **Backend / Base de Datos:** Google Apps Script (para recibir peticiones `POST` desde el formulario) y Google Sheets.
- **Comunicaciones:** API de WhatsApp (`wa.me`).
